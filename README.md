# DamagedMilitary

Recognition of damaged military equipment.
***

## Usage
### ReactJS dependencies
```bash
cd frontend
npm install --from-lock-json
npm audit fix
```
***

### Containerization
```bash
docker-compose up -d
```
***

App is now running at `localhost:3000`
